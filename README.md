A 2D tic tac toe game:

  1. When you enter the game both players  enter their names (a message will be sent).
  2. the game will present that player 1 symbol is an X and player 2 symbol is O.
  3. there will be an explanation printed on the screen how to enter the indexes to play the game with a board-game map.
  4. start entering the indexes.
  5. the screen will print out each time whose turn it is.
  6. each turn we check if the index is available. if it isnt you are requested to re-enter a new index.
  7. each turn we also check if the player won the game. if you won an appropriate messege will be sent out.
  8. if the board is full (after the 9th turn) a messege will be sent out that the board is full
